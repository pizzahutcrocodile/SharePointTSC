How to Grant Permissions!

Click on the Gear Icon named "settings".

Click on the Link Named "Site permissions".

Click on the Link named "Advanced permissions settings".

Click on the button named "Grant Permissions".

Click on the text box and search for a user you'd like to add.
If it is possible to add the user the software should show you options as you finish typing the name of the user.

Hover over the Permission Group you'd like to add the user to and click to select.

Click the Share Button.
